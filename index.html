Learning Task-Centric World Models from Visual Foundations

Turning frozen foundation embeddings into compact, planning-relevant latent dynamics for reward-free offline visual planning.

OpenReview Code

Abstract

World-model latents are typically either learned from pixels with limited semantic structure or inherited from frozen visual foundation models with excessive task-irrelevant detail — both poorly matched to downstream planning. We propose TC-WM, which treats the pretrained embedding space as a semantic scaffold rather than the final state space. TC-WM linearly projects high-dimensional visual embeddings into a compact latent, aligns a designated subspace with the agent's physical state via contrastive learning, and reconstructs embeddings to preserve useful visual structure. Theoretically, TC-WM identifies the task-centric latent factors up to an invertible transformation. Empirically, TC-WM enables zero-shot planning across nine offline benchmarks spanning navigation, locomotion, and manipulation, achieving stronger world-model quality and more precise control than state-of-the-art baselines.

Overview

Foundation embeddings encode textures, lighting, and backgrounds — not control. When used directly as a world-model latent (e.g., DINO-WM, V-JEPA), the model spends capacity on irrelevant variation. The cost is acute under high-dimensional action: Maze uses 2-D actions, but Robomimic requires a 7-DoF arm with 43-D proprioception, where every irrelevant latent dimension is wasted planning capacity. TC-WM treats the embedding as a scaffold: a designated latent subspace is aligned with proprioception via contrastive learning, while the complementary subspace stays anchored to the foundation manifold through embedding reconstruction. The result is a compact latent that is both physically grounded and predictively sufficient.

Method

Architecture. A compact, task-centric latent is extracted from frozen visual embeddings using proprioception as side information.

The pipeline has six trainable components on top of a frozen visual backbone (e.g., DINOv2):

  • Joint Embedding — concatenate the frozen visual feature with a trainable proprioception embedding.
  • Latent Encoder — a linear projection to a compact latent.
  • TC Alignment — an InfoNCE contrastive loss aligns a designated subspace with proprioception; an L1 penalty selects the task-centric block by sparsity.
  • Latent Dynamics — a ViT transition predicts the next latent from history.
  • TC Dynamics — an auxiliary head predicts the next proprioception, enforcing task-centric temporal structure.
  • Embedding Decoder — a linear decoder reconstructs the visual embedding, anchoring the latent to the foundation manifold and preventing collapse.

Under standard contextual-dynamics and partial-alignment assumptions, the learned latent is block-wise identifiable and the task-centric block is recovered up to an affine map — the linear projection plus contrastive alignment thus produces a compact, physically grounded latent, not merely a predictively accurate one.

Environments

We evaluate on nine offline visual-control environments spanning three task families:

  • Navigation — Maze and Wall (D4RL Point-Maze, 2-D actions).
  • Manipulation — Push-T (2-D pusher) and Robomimic's Lift, Can, and Square (7-DoF arm, up to 43-D proprioception).
  • Locomotion — Reacher, Cheetah, and Hopper from the DeepMind Control Suite.

All methods are trained on the same offline trajectories of images, actions, and proprioception — no reward, no online interaction.

Results

We compare against TD-MPC2 (compact latent dynamics), DreamerV3 (recurrent generative latent), MuZero (task-oriented latent, reproduced offline), and DINO-WM (the closest embedding-space baseline), all trained on the same offline trajectories.

World-model prediction. TC-WM achieves the lowest latent-prediction error on nearly all tasks while remaining competitive on image reconstruction. PR and LP, lower is better, 300k training steps.

Planning. CEM on Maze / Wall / Push-T (success rate) and Cheetah / Hopper (return); LDP on Lift / Can / Square. TC-WM matches the strongest baseline on saturated CEM tasks and is the only method that surpasses DINO-WM on every LDP task.

Ablation

Architecture / projection ablations. Random projection fails — dimensionality reduction alone is not enough. DINOv2 and DINOv3 dominate as foundation encoders. The linear projection head is best, consistent with affine-identifiability.

Loss components on Lift. Removing embedding reconstruction collapses both planning and visual fidelity; removing proprioceptive supervision lowers success rate while preserving SSIM; the latent split dimension matters little.
